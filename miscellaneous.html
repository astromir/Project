  • Tags Used

    Tag Purpose
    <!DOCTYPE> Defines the document type and version of HTML being used.
    <html> Defines the root element of the HTML document.
    <head> Contains meta-information about the HTML document.
    <meta> Provides metadata about the HTML document.
    <title> Specifies the title of the HTML document.
    <link> Used to link external resources.
    <body> Contains the content of the HTML document.
    <header> Represents introductory content or navigation links.
    <img> Embeds an image in the document.
    <div> Defines a division or section in the document.
    <h1>, <h3> Headings of varying levels.
    <p> Defines a paragraph of text.
    <button> Defines a clickable button element.
    <a> Creates hyperlinks to other web pages or resources.
    <ul>, <li> Creates an unordered list and list items.
    <hr> Represents a thematic break or separator.
    <footer> Defines a footer for a document or section.
    <form> Defines a form for user input.
    <label> Defines a label for an input element.
    <input> Defines an input control.
    <table> Defines a table structure.
    <thead> Defines the header of a table.
    <tr> Defines a row in a table.
    <th> Defines a header cell in a table.
